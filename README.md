# IP Look Up

Bu loyiha `Vite + React` asosiga ko'chirildi va dependency'lar latest barqaror versiyalarga yangilandi.

## Ishga tushirish

```bash
npm install
npm run dev
```

Brauzerda odatda `http://localhost:5173` ochiladi.

## Build

```bash
npm run build
```

Build natijasi `dist/` papkaga chiqadi.

## Asosiy stack

- React 19
- Vite 8
- React Router 7
- React Leaflet 5
- Bootstrap 5
- Axios
